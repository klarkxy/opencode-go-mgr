# OpenCode-Go 防滥用声明

OCG Manager 只用于合法的本地账号管理和 Gateway 路由。它不是用于滥用 OpenCode-Go、规避限制、批量养号、绕过验证或隐藏未授权使用的工具。

请勿使用本项目：

- 自动注册账号、自动过验证码、自动充值或自动完成身份校验；
- 绕过 OpenCode-Go 的额度、冷却、计费、限速或访问控制；
- 聚合不属于你、或你无权使用的 key；
- 以违反 OpenCode-Go 条款的方式转售共享 Gateway 访问；
- 运行垃圾信息、爬取、撞库、欺诈或其他滥用流量；
- 将自动化滥用伪装成本地客户端的正常请求。

OCG Manager 会记录上游 429 冷却时间，并跳过冷却中的账号，因为尊重上游限制是项目设计的一部分。远端同步只应用于用户自己控制的节点，不能用于收集未授权 key。

使用者有责任遵守 OpenCode-Go 的服务条款、当地法律，以及所访问上游服务的规则。维护者可以拒绝支持疑似滥用的工作流。
